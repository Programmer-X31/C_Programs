# C projects

## Compilation - Manual

### Normal

To compile a C program, run the command:

```
gcc *file.c* -o *file.exe*
```

### Math.h

To compile a C program with Math.h, run the command:

```
gcc -lm *file.c* -o *file.exe*
```

## Compilation - Automatic

```
./run.sh *source.c* *exec.exe*
```
